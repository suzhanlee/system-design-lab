## Phase별 기록

### Phase A: Blank Canvas

**사용자 작성 내용**:
```
프로젝트명: 긴 url -> short url 로 변경하는 프로젝트

비즈니스 목표:
1. 긴 url 을 짧은 url 로 변경해주는 서비스
2. 사용자가 긴 url을 입력하면 고유한 단축 url 을 생성하고, 해당 단축 url 접속 시 원본 url 로 라우팅

사용자 페르소나:
- 주 사용자: url이 불필요하게 길어 짧고 직관적으로 이해가 가는 url로 변경하는 기능을 만들고 싶은 사람

핵심 기능:
1. 긴 url을 짧은 url로 만들어야 한다.
2. 이때 짧게 만든 url로 접속 시 긴 url로 자동 라우팅 되게 구성해야 한다.

기술적 제약:
- java/spring/jpa를 사용해서 개발한다.
```
