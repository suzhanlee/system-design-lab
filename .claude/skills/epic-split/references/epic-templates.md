## epic-roadmap.md

```markdown
# Epic 로드맵

## 구현 순서

```
Epic 1 (기반)
   ↓
Epic 2 → Epic 3 (병렬 가능)
   ↓
Epic 4 (통합)
```

## 상세 일정

| 순서 | Epic | 의존성 | 예상 소요 |
|------|------|--------|----------|
| 1 | Epic 1 | - | 1h |
| 2 | Epic 2 | Epic 1 | 1h |
| 3 | Epic 3 | Epic 1 | 1h |
| 4 | Epic 4 | Epic 2, 3 | 1h |

## 마일스톤
- **M1**: Epic 1 완료 → [기능 설명]
- **M2**: Epic 2, 3 완료 → [기능 설명]
- **M3**: Epic 4 완료 → [최종 기능]
```

## Epic 크기 기준

| 항목 | 권장 범위 |
|------|----------|
| Entity | 1~2개 |
| 기능 | 1개 CRUD 스트림 |
| 코드 파일 | 10~20개 |
| 테스트 | 5~10개 시나리오 |
| 소요 시간 | 약 1시간 |

## 완료 루브릭

각 Epic은 다음을 포함해야 함:
- [ ] 제목 (Korean)
- [ ] 범위 (포함/제외 기능)
- [ ] Entity 목록
- [ ] 완료 기준 (DoD)
- [ ] 의존 Epic (있으면)
