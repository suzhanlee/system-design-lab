```markdown
## ADR Creation Guide
1. 새 ADR 번호 할당 (순차적 증가)
2. **Phase A: Pre-Mortem 질문 답변**
3. **Phase B: Trade-off Matrix 작성 (최소 3개 대안)**
4. Phase C: 템플릿 기반 ADR 본문 작성
5. **Phase D: Self-Critique 수행 (평균 4점 이상 달성)**
6. `/redteam` 실행하여 비평 수행
7. 비평 반영 후 Status: Accepted로 변경
```
